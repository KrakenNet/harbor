# How to Author a Bosun Pack

Bosun packs bundle skills, prompts, and policy assets into a single distributable unit.

## Outline

1. Lay out the pack directory (`skills/`, `prompts/`, `policies/`, `manifest.toml`).
2. Declare a `harbor.packs` entry point.
3. Register provenance metadata so traces can attribute outputs back to the pack.
