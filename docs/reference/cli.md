# CLI Reference

The `harbor` CLI drives graph execution, replay, and plugin introspection.

## Subcommands (planned)

- `harbor run <graph.yaml>` — compile and execute a graph.
- `harbor replay <run-id>` — re-execute a recorded trace.
- `harbor plugins` — list discovered plugins and their `api_version`.
- `harbor compile <graph.yaml>` — emit the IR without running.
